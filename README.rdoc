<b>Задача</b>:

Добавить в модель Product метод, возвращающий 6 наиболее похожих товаров.

* Степень "похожести" определяется по количеству совпадений значений свойств следующих типов: UID_CHOICE = 4, UID_INTEGER = 2, UID_BOOLEAN = 5. У какого товара больше совпадений, тот и более похож, все типы равноценны.
* Значения свойств хранятся в таблице product_properties. Список свойств, значения которых учитываются, берется из таксона: taxon.properties.
* Дамп БД в файле test_case.sql.
* Ожидаемое реальное количество товаров в выборке > 100 000.
